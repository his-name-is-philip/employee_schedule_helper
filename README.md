# employee_schedule_helper

## Описание
Набор макросов VBA для Excel, который упрощает пользование графиками перерывов для сотрудников банка. Данные о перерывах загружаются супервайзерами в определённую папку в утвержденном формате. Макрос отображает информацию о перерывах в удобном виде за несколько кликов.

## Возможности
- Автоматическая загрузка файлов с расписаниями при открытии файла
- При наличии нескольких подходящих файлов открывается диалоговое окно для выбора файла.
- Обновление отображаемого графика без перезапуска файла.
- Отображение имени исходного файла рядом с кнопкой «Обновить» для удобства.


## Требования
- Microsoft Excel с поддержкой макросов (xlsm).
- Разрешение на запуск макросов (см. раздел «Устранение неполадок»).

## Установка
1. Скачайте файл `Breaks_helper.xlsm` или клонируйте файл с макросом:  
   ```cmd
   curl -L -o Breaks_helper.xlsm https://raw.githubusercontent.com/his-name-is-philip/employee_schedule_helper/main/Breaks_helper.xlsm
2. ПКМ по файлу → Свойства → внизу (если есть) поставить галочку Разблокировать → ОК.
3. Откройте файл в Excel. При появлении предупреждения нажмите **Включить содержимое** (Enable Content).
4. Перейдите на лист Настройки и внесите свои ФИО и другие обязательные параметры.
5. Сохраните файл: `Ctrl+S` или Файл → Сохранить

При запуске файл от супервайзеров подгружается автоматически, если папка содержит файл только на одну дату. Если папка содержит несколько файлов, соответствующих критериям поиска, на экране отобразится окно выбора файла из подходящих - открывайте файл с нужной датой.
Чтобы обновить график на актуальный, не перезапуская файл, нажмите "Обновить".
Для удобства между графиками и кнопкой "Обновить" расположено название исходного файла с графиками.

## Использование
* При запуске: если в папке с файлами графиков находится только один файл, соответствующий критериям, он загружается автоматически.
* Если в папке несколько подходящих файлов — появится диалоговое окно: откройте файл с нужной датой.
* Чтобы обновить данные в уже открытом файле, нажмите кнопку Обновить. Имя исходного файла отображается рядом с кнопкой для удобства.
> Примечание: расположение папки с исходными файлами, параметры для поиска по названию файла и ФИО задаются на листе Настройки. Заполните лист с настройками перед использованием.
## Устранение неполадок
* Если Excel не позволяет запускать макросы:
  * Откройте «Файл → Параметры → Центр управления безопасностью → Параметры центра управления безопасностью → Параметры макросов» и выберите подходящий режим (рекомендуется — «Отключить все макросы с уведомлением» и вручную включать содержимое для доверенных файлов).
  * Проверьте, что файл не заблокирован системой (см. Установка, шаг 2).
* Если вы получаете ошибки при загрузке — проверьте:
  * соответствие пути к папке шаблону:
  `\\folder.topleveldomain\folder\Schedul files\`
  * наличие строки с вашим именем на указанном листе 
  * точное соответветствие ФИО, указанного в настройках, содержанию ячейки с ФИО в исх. файле
  
* Рекомендуется подписать макрос цифровой подписью для упрощения развёртывания в корпоративной среде.
